## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm`/`yarn` installed globally on your machine.  

Installation:

`npm install` or `yarn install`  

To Start Server:

`npm dev` or `yarn dev`  

To Visit App:

`localhost:3000`
